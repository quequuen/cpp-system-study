# C++ System Study

C++ 시스템 프로그래밍 개인 스터디 프로젝트입니다.

## 프로젝트 소개

이 프로젝트는 C++ 시스템 프로그래밍을 학습하기 위한 개인 스터디 저장소입니다.
메모리 관리, 프로세스/스레드, 파일 시스템, 네트워크 프로그래밍 등 시스템 레벨의 개념들을
실습을 통해 학습합니다.

## 학습 목표

- C++ 언어의 고급 기능 이해
- 시스템 프로그래밍 개념 습득
- 메모리 관리 및 최적화
- 멀티스레딩 및 동시성 프로그래밍
- 파일 I/O 및 네트워크 프로그래밍

## 요구사항

- C++17 이상 지원 컴파일러 (GCC, Clang, MSVC)
- CMake 3.10 이상 (선택사항)

## 빌드 및 실행

### 컴파일

```bash
# 직접 컴파일
g++ -std=c++17 -o program src/main.cpp

# 또는 CMake 사용
mkdir build && cd build
cmake ..
make
```

### 실행

```bash
./program
```
